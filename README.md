# 🍽️ NutriDapur

**NutriDapur** adalah aplikasi web resep masakan bernutrisi yang dirancang untuk membantu pengguna memilih resep makanan berdasarkan kebutuhan gizi harian dan kondisi kesehatan tertentu. Aplikasi ini dibangun menggunakan **Laravel** untuk backend, **React JS** untuk frontend, serta **Inertia.js** untuk menghubungkan keduanya dalam arsitektur SPA (Single Page Application). Styling menggunakan **Tailwind CSS** agar lebih responsif dan modern.

🔗 **Live Demo**: [https://nutrixplore-production.up.railway.app](https://nutrixplore-production.up.railway.app)  
🎨 **Desain UI (Figma)**: [Klik di sini](https://www.figma.com/design/ASmqkmhc90Ng1UFlGALBV9/Perancangan-TA?node-id=0-1&t=u4HcvCsKgq65Xlfe-1)

---

## ✨ Fitur Utama

- 🔍 Jelajahi resep berdasarkan filter nutrisi dan preferensi makanan
- 🧠 Artikel dan edukasi terkait kesehatan dan gizi
- ❤️ Simpan resep favorit (jika login)
- 🖋️ Editor untuk membuat resep (untuk admin)
- 📱 Desain responsif dan siap untuk PWA (offline mode)

---

## ⚙️ Teknologi yang Digunakan

- **Laravel** v12.1.1 – Backend dan routing
- **React JS** v19.0 – Antarmuka pengguna
- **Inertia.js** – Penghubung Laravel dan React
- **Tailwind CSS** v4.0 – Styling modern
- **PHP** v8.3.12
- **Node.js** v22.9.0
- **MySQL** v8.0.30

## 💻 Cara Instalasi & Menjalankan Proyek

### 1. Clone repositori
```bash
git clone https://github.com/vinaanurfauziah/nutridapur.git
cd nutridapur 
````
### 2. Install dependencies
```bash
npm install
````
### 3. Jalankan server lokal
```bash
npm run dev & php artisan serve
````
### 4. Akses aplikasi
```bash
Buka di browser: http://127.0.0.1:8000/
